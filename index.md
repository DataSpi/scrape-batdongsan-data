## What this project does

- Scrape real estate listing and project data from batdongsan.com.vn.
- Store and transform data in PostgreSQL/Supabase.
- Model and analyze with Malloy + dbt.
- Publish report outputs for data exploration.

For full details, see [readme.md](readme.md).
